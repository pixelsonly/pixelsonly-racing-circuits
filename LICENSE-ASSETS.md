# Visual asset licenses

Visual assets are licensed per asset, because they come from different sources.
This file is the authoritative register — **every flag and any third-party
geometry must have a row here before it is committed.**

## Track map SVGs

Track outline SVGs generated by Pixelsonly Racing are © 2026 Pixelsonly Racing,
released under **CC BY-SA 4.0** (`CC-BY-SA-4.0`), consistent with the narrative
content. If a map is traced from or derived from a third-party geometry source,
that source's terms apply and must be recorded in the **Track maps** register
below.

## Track maps

Track-map SVGs in this repo are hand-traced single-line outlines authored by
Pixelsonly Racing, but the underlying geometry is typically derived from a
public-domain or CC-BY-SA Wikimedia Commons reference. Record every traced map
here — the source file, its license, and the date it was fetched — so the
derivative's obligations are unambiguous. `npm run fetch-map` writes rows into
this table automatically.

| Track / map | Source | License | Notes |
|---|---|---|---|

## Satellite imagery derivatives

Published `satellite.*` derivatives are stylized works derived from **Copernicus
Sentinel-2** data. They require the attribution **"Contains modified Copernicus
Sentinel data <year>"** wherever shown or sold. The raw Sentinel-2 masters are
**not** part of this repo (they live in a private bucket and are never published).
